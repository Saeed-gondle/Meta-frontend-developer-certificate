Introduction to HTML and CSS!

Core technologies of web development

Technology 1
HTML - HyperText Markup Language is the standard markup language for creating web pages and web applications. It describes the structure and content of web documents.
Technology 2
CSS - Cascading Style Sheets is used for describing the presentation of a document written in HTML. It controls the layout, colors, fonts, and overall visual appearance.
Technology 3
JavaScript - A programming language that enables interactive web pages and dynamic functionality. It brings websites to life with user interactions and real-time updates.
Technology 4
React - A JavaScript library for building user interfaces, particularly single-page applications. It makes creating complex, interactive UIs more manageable.
